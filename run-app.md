# 運行應用指令

## Android 模擬器/設備
```bash
npm run android
```

## iOS 模擬器/設備 (需要 macOS)
```bash
npm run ios
```

## 預覽模式 (在瀏覽器中預覽)
```bash
npm run preview
```

## 清理項目
```bash
npm run clean
```
