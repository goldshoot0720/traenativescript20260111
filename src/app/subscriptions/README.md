# 訂閱管理組件

## 概述
訂閱管理組件已經修改為類似食品管理的結構和功能，提供更一致的用戶體驗。

## 主要功能

### 1. 訂閱清單管理
- 顯示所有訂閱服務
- 支持點擊查看詳細資訊
- 提供編輯和刪除功能
- 顯示價格、下次付款日期、類別等資訊

### 2. 新增訂閱
- 完整的表單輸入
- 支持快速模板（Netflix、Spotify、iCloud）
- 包含類別和描述欄位
- 表單驗證和清除功能

### 3. 搜索和篩選
- 按名稱搜索訂閱
- 按類別篩選
- 查看即將到期的訂閱

### 4. UI 改進
- 採用與食品管理相同的 GridLayout 結構
- 統一的標籤頁導航
- 一致的樣式和交互模式
- 響應式設計支持

## 技術改進

### 1. TypeScript 更新
- 添加 `OnDestroy` 生命週期
- 擴展 `SubscriptionItem` 介面
- 添加連接狀態管理
- 改進錯誤處理

### 2. HTML 結構
- 從 TabView 改為 GridLayout
- 統一的行動裝置優先設計
- 改進的表單佈局
- 更好的空狀態處理

### 3. SCSS 樣式
- 與食品管理保持一致的樣式
- 改進的響應式設計
- 統一的顏色方案和間距
- 更好的視覺層次

## 使用方式

### 查看訂閱清單
1. 點擊「📋 訂閱清單」標籤
2. 瀏覽所有訂閱服務
3. 點擊項目查看詳細資訊
4. 使用右上角的功能按鈕進行搜索、篩選等操作

### 新增訂閱
1. 點擊「➕ 新增訂閱」標籤
2. 填寫服務資訊
3. 可使用快速模板快速填入常見服務
4. 點擊「新增訂閱」完成

### 編輯訂閱
1. 在清單中點擊項目右側的「⋯」按鈕
2. 選擇「編輯」
3. 系統會自動切換到新增頁面並填入現有資料
4. 修改後重新新增即可

## 資料結構

```typescript
interface SubscriptionItem {
    id: string;
    name: string;
    price: number;
    account: string;
    site: string;
    nextDate: string;
    note?: string;
    category?: string;
    description?: string;
    createdAt?: string;

```
