# 食品管理系統重構

## 重構內容

根據提供的設計圖片，已將食品管理界面重構為現代化的桌面應用程式風格：

### 主要改進

1. **側邊導航欄**
   - 添加了鮮元AI系統品牌標識
   - 完整的導航菜單，包含各個功能模塊
   - 紫色主題配色

2. **主內容區域**
   - 分為左右兩欄布局
   - 左側：食品清單展示
   - 右側：新增食品表單

3. **食品清單**
   - 卡片式設計，每個食品項目獨立顯示
   - 包含產品圖片、名稱、數量、價格、到期日
   - 刪除按鈕便於管理

4. **新增表單**
   - 整合所有必要欄位
   - 清晰的標籤和輸入框
   - 綠色新增按鈕

### 技術特點

- 使用 NativeScript GridLayout 實現響應式布局
- SCSS 樣式文件提供一致的視覺設計
- TypeScript 組件邏輯保持原有功能
- 支援圖片上傳和數據驗證

### 顏色主題

- 主色調：紫色系 (#6a4c93, #8b5cf6)
- 強調色：綠色 (#00e676) 用於操作按鈕
- 警告色：紅色 (#e74c3c) 用於到期日和刪除

### 使用方式

1. 左側導航選擇功能模塊
2. 主區域查看和管理食品清單
3. 右側表單新增食品項目
4. 支援同步功能更新數據
